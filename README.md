# Interactive Quiz Game

A modern, responsive, and visually appealing Quiz Game application built using **HTML, CSS, and JavaScript**. This project features multiple question types, smooth animations, a pastel-themed user interface, and dynamic score calculation, providing an engaging learning experience while demonstrating core front-end development skills.

---

## Features

* Single Select (Multiple Choice) Questions
* Multi Select Questions
* Fill in the Blank Questions
* Dynamic Progress Bar
* Automatic Score Calculation
* Restart Quiz Functionality
* Responsive Design for All Devices
* Modern Glassmorphism UI
* Soft Pastel Color Palette
* Smooth Hover Effects and Transitions
* Built with Pure HTML, CSS, and JavaScript
* No External Libraries Required

---

## How to Run

1. Clone this repository.
2. Navigate to the project folder.
3. Open `index.html` in your preferred web browser.
4. Start answering the quiz questions.

No installation or additional dependencies are required.

---

## How It Works

* The application displays one question at a time.
* Users answer different types of questions:

  * Single Select
  * Multi Select
  * Fill in the Blank
* A progress bar updates as the quiz advances.
* After the final question, the application calculates and displays the user's score.
* Users can restart the quiz at any time.
